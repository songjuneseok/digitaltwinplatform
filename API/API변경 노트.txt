	*5002 port
		>모터 수식 바꾸기(모터정보 위치 찾기)
			1)poll, hertz, watt, torque 받아서 모터곡선 확인
				- full load = poll * hertz
 
		> P,Q,R중 2개의 파라미터 받은 후 X, Y, Val, mp,me,fp,fe 반환
			1)P,Q,R 연산완료.
			2)RPM으로 모터의 효율 계산
			3)Q로 팬의 파워와 효율 계산 
			4)3에서 계산한 파워에 모터의 효율을 곱해서 모터 파워 계산
		>단위변환함수 삽입.
			1)계산된 값에 따른 단위를 쿼리로 찾음
			2)단위환산 후 json으로 보냄.
